# Web-Scraping

## Installation

```git clone https://github.com/b2r2/Web-Scraping.git```

### Run application
```python3 main.py```

This is simple application a Web-Scraping only for sites [telegraph](http://telegra.ph/) or [zen](http://zen.yandex.ru)
